# 游戏设计专业技能集

本技能集基于专业游戏设计书籍生成，涵盖了游戏设计的核心理论、心理学应用、机制平衡、关卡构建、经济系统、UI/UX设计以及项目管理等多个维度。这些技能旨在帮助游戏设计师、开发者及相关人员在游戏开发的各个阶段进行决策、分析和优化。

## Available Skills

| Skill | Description | Use When |
|-------|-------------|----------|
| [player-gratification-classification](player-gratification-classification/SKILL.md) | 分析玩家属于即时满足型还是延迟满足型，并根据不同类型特征设计相应的游戏机制、平衡策略和游戏化元素 | 游戏类型设计、游戏平衡调试、游戏化元素设计 |
| [game-economy-supply-demand-design](game-economy-supply-demand-design/SKILL.md) | 设计游戏内经济系统的供需平衡与资源交换机制 | 构建多层次物品体系、处理供需波动、设计时间与金钱交换机制 |
| [game-design-frameworks](game-design-frameworks/SKILL.md) | 使用核心游戏设计框架（拉扎罗的4种趣味元素、魔法圈、MDA框架、记忆/技巧游戏分类）分析游戏系统和玩家体验 | 分析玩家动机、设计游戏机制、评估游戏结构、分类游戏类型 |
| [visual-spatial-psychology-design](visual-spatial-psychology-design/SKILL.md) | 应用视觉与空间心理学原理设计游戏场景、界面布局和建筑比例 | 关卡场景设计、UI布局、建筑建模、调节玩家情绪、引导视线注意力 |

| [player-learning-error-handling](player-learning-error-handling/SKILL.md) | 设计玩家学习曲线、错误分类体系和失败处理机制，平衡挑战性与挫败感 | 新手引导设计、技能系统设计、任务难度规划和失败状态设计 |

| [arcs-achievement-design](arcs-achievement-design/SKILL.md) | 应用ARCS模型设计具备激励效果的教学关卡、成就系统或任务系统 | 教学关卡设计、成就系统设计、任务系统设计 |
