# 锻造系统案例

## Minecraft 合成系统

**特点**：
- 网格化配方
- 直观的材料摆放
- 大量组合可能性

**成功因素**：
- 逻辑清晰（木棍+木板=工具）
- 实验乐趣
- 社区分享

## 生化危机 草药合成

**特点**：
- 简单的3槽系统
- 红+绿=混合草药
- 红+蓝=强化草药

**附加价值**：
- 节省背包空间
- 资源管理策略
- 生存压力下的决策

## 技能连招设计

**基础连招**：
```
轻击 × 2 + 重击 = 浮空攻击
```

**进阶连招**：
```
冲刺 + 轻击 + 跳跃 + 重击 = 空中连击
```

**终极连招**：
```
(轻击 × 3) + (重击 × 2) + 特殊键 = 必杀技
```

## 协同效应公式

```
协同效果 = (元素A效果 + 元素B效果) × 协同系数

其中协同系数通常为 1.2 - 1.5
```
