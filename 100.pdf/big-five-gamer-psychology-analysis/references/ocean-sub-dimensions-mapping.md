# 大五人格子维度与游戏偏好映射表

## 1. 开放性
子维度包括：想象力、艺术趣味、情绪性、冒险性、求知欲、自由主义。

**映射示例：**
- **想象力**：
  - 高分：偏爱奇幻、奇异世界、超现实设定。
  - 低分：偏爱现实、写实、历史题材。
- **冒险性**：
  - 高分：偏爱探索未知、尝试新事物、随机性高的机制。
  - 低分：偏爱建造、种地、明确边界和规则的内容。

## 2. 尽责性
子维度包括：能力、条理性、尽责、追求成就、自律、深思熟虑。

**映射逻辑：**
- 高尽责性：偏好任务导向、清晰的进度系统、成就收集、规划类玩法。
- 低尽责性：偏好自由探索、无明确目标、休闲随性的玩法。

## 3. 外倾性
子维度包括：热情、乐群、独断、活力、寻求刺激、积极情绪。

**映射逻辑：**
- 高外倾性：偏好多人社交、团队合作、竞技对抗、热闹的社交环境。
- 低外倾性：偏好单人内容、深度沉浸、独立探索、小团体协作。

## 4. 随和性
子维度包括：信任、坦诚、利他、顺从、谦逊、同理心。

**映射逻辑：**
- 高随和性：偏好PVE合作、帮助他人、非竞争性社交、和谐的游戏氛围。
- 低随和性：偏好PVP竞争、对抗、权力争夺、个人主义玩法。

## 5. 神经质
子维度包括：焦虑、愤怒敌意、抑郁、自我意识、冲动性、脆弱性。

**映射逻辑：**
- 高神经质：对高难度挑战敏感，可能偏好逃避现实或情感宣泄类内容，需注意挫败感管理。
- 低神经质：能从容应对高压环境，偏好硬核挑战或需要冷静判断的策略内容。
