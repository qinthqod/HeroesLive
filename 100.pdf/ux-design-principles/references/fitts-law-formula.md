# 菲兹定律公式

## 数学公式

```
T = a + b × log2(D/W + 1)
```

**参数说明**：
- T：到达目标所需时间
- a：起始时间（通常可忽略）
- b：设备相关常数
- D：起始点到目标中心的距离
- W：目标的宽度

## 实际应用

### 按钮尺寸计算

给定距离D=200像素，期望时间T=0.5秒：
```
0.5 = log2(200/W + 1)
2^0.5 = 200/W + 1
1.414 - 1 = 200/W
W = 200 / 0.414
W ≈ 483 像素
```

### 设计启示

**距离影响**：
- 距离增加2倍，时间增加约1倍
- 距离增加4倍，时间增加约2倍

**宽度影响**：
- 宽度增加2倍，时间减少约0.5倍
- 宽度增加4倍，时间减少约1倍

**边缘优势**：
- 屏幕边缘的目标可视为无限宽
- 菜单栏放在屏幕边缘更易点击
