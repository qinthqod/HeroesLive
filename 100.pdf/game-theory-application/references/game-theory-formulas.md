# 博弈论公式详解

## 纳什均衡计算

给定得益矩阵，纳什均衡的判定条件：

```
对于单元格 (i, j):
- 第一个值是该列所有行的最大值
- 第二个值是该行所有列的最大值
```

## 逆向归纳法步骤

1. 从决策树的末端（最终结果）开始
2. 在每个决策节点，选择对该玩家最优的分支
3. 标记非最优分支为“不会选择”
4. 向上回溯到前一个决策节点
5. 重复步骤2-4直到根节点

## 帕累托改进判定

```
IF (状态A → 状态B) AND
   (至少一人收益增加) AND
   (无人收益减少)
THEN 状态B是状态A的帕累托改进
```

## 囚徒困境得益矩阵示例

```
          玩家B
        合作    背叛
合作  (3,3)  (0,5)
背叛  (5,0)  (1,1)
```

- (背叛,背叛) 是纳什均衡
- (合作,合作) 是帕累托最优
- 理性选择导致非最优集体结果
