# 零和博弈判定与策略

## 触发条件

当出现以下情况时应用此技能：
- 游戏系统的总得益可能恒为零（赢=输）
- 存在多方互动和明确收益损失计算
- 需要设计竞技平衡机制

## 判定逻辑

**核心公式：**
获胜方收益 + 失败方损失 = 0

**特征：**
- 一方的收益必然对应另一方的损失
- 所有结果都是"帕累托最优"

**典型案例：**
- 扑克牌游戏（赢的钱=输的钱）
- 石头剪刀布（每局总得益为0）

**反例：**
囚徒困境（双方可能同时赢或输，刑期之和不为零）

## 解决策略

### 混合策略
- 纳什均衡：寻找博弈的平衡点
- 混合策略：在一系列游戏中使用随机选择

### 示例（石头剪刀布）
随机选择石头、剪刀或布：
- 结果概率：1/3赢，1/3平，1/3输
- 效果：平均最坏情况优于单局最坏情况
- 目标：提高最小得益率

### 极小极大策略
旨在增加玩家所能获得的最小得益。

## 理论依据

约翰·冯·诺伊曼和奥斯卡·摩根斯坦的研究表明：
每一个零和博弈都有一个"极小极大"解决方案。
即使纳什均衡不存在，基于随机选择的混合策略也能提高最小回报。

## 输出要求

- 博弈类型判定结果
- 建议采用的平衡策略（如混合策略）
- 具体实施参数
