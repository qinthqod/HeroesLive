# 博弈论应用示例

## 零和博弈典型场景

### 扑克游戏
- 玩家A赢得100筹码
- 玩家B失去100筹码
- 总得益：100 + (-100) = 0

### 石头剪刀布
| 玩家A \ 玩家B | 石头 | 剪刀 | 布 |
|---------------|------|------|-----|
| 石头 | 0,0 | 1,-1 | -1,1 |
| 剪刀 | -1,1 | 0,0 | 1,-1 |
| 布 | 1,-1 | -1,1 | 0,0 |

## 非零和博弈反例

### 囚徒困境
- 双方都沉默：各判1年（总刑期2年）
- 双方都背叛：各判5年（总刑期10年）
- 一方背叛：背叛者释放，沉默者判10年

总得益不为零，不是零和博弈。

## 混合策略计算

对于2x2零和博弈，最优混合策略概率：
p = (d-c) / (a+d-b-c)

其中收益矩阵为：
| | 对手策略1 | 对手策略2 |
|-------|-----------|-----------|
| 我方策略1 | a | b |
| 我方策略2 | c | d |
