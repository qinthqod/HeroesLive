# Image-2 素材设计方向

本项目的视觉目标是“凡人修仙早期小人物冒险”而不是仙侠大场面：画面要有低阶修士的朴素、危险和资源匮乏感，适配《月圆之夜》式单局推进、卡牌选择、遭遇战和奖励事件。

## 统一风格

- 画风：半写实东方奇幻卡牌插画，细节克制，墨色边缘，低饱和玉青、朱砂、旧金、夜蓝作为主色。
- 构图：卡牌图优先使用中心主体，背景只交代地点或术法余波，避免复杂群像。
- 可读性：移动端也能看清主体轮廓；不要大面积文字，不要水印，不要现代服饰、现代建筑、科幻元素。
- 世界观限制：主角处于炼气到筑基阶段，素材应体现“符箓、灵石、草药、低阶法器、山谷、坊市、妖兽残影”等早期修仙元素。
- 升级表现：升级牌用同一张卡图即可，UI 上以 `+`、边框亮度和数值变化体现“精研”，不要为首版每个升级状态单独生成新图。

## 已实现玩法素材钩子

- 法宝系统已接入主循环：战斗奖励中可出现法宝，与拿牌和调息形成选择。
- 首批法宝：月华玉佩、狼牙剑坠、青竹残简、聚灵灯、坊市木牌。
- 法宝首版先使用 UI 文字牌呈现；后续如生成图标，应保持“低阶法器、小而实用、材质朴素”的方向，不做神器级视觉。
- 可优先补 1:1 小图标：玉佩、狼牙坠、竹简、旧灯、木牌，用于奖励按钮和顶部法宝栏。
- 心魔负面牌已接入敌人与事件：心魔杂念不可打出，会堵塞手牌；散修、黑煞教执事和强破禁制事件会把它混入牌堆。
- 心魔类素材如后续生成，应表现为破碎墨痕、杂念符灰、暗色灵台裂纹，避免怪物实体化，保持“内耗和污染牌组”的读感。
- 战斗状态已接入：燃烧会在玩家回合开始灼伤敌人，虚弱会强化下一次受击或承受敌人下一次重击。
- 状态图标后续可补朱砂火痕和紊乱灵息两枚小图标，用于敌人名牌与顶部状态栏。
- 敌方意图已接入承伤预测：伤害招式会按当前护盾、玩家虚弱和多段攻击计算预计失血、剩余护盾与虚弱消耗。
- 战斗日志已扩展结算反馈：敌人行动会记录护体、破绽、实际威力、挡伤、失血和虚弱消耗；消耗品会记录恢复、灵气、护盾、抽牌、驱散和伤害结果。
- 悬停提示已接入：卡牌、路线、事件、奖励、消耗品、流派、劫数和局内小按钮都会显示说明；手牌不可打出时会提示灵气不足或心魔不可打出。
- 地图节点预估已接入：路线按钮悬停会提示普通战、精英、Boss、事件、坊市、调息和修炼的预期收益、敌人压力、当前血线/灵石/心魔/灵气契合度，帮助进节点前判断风险。
- 手牌动态预估已接入：战斗中悬停手牌会预估灵气变化、威力、破盾、伤血、剑势消耗、护盾、恢复、燃烧、虚弱、净心和抽牌结果，减少心算负担。
- 敌情研判已接入：敌方面板会概括敌人招式循环的最高威力、多段、护体、破绽、虚弱和心魔污染，并按当前手牌、护盾与血线给出一句行动计划和本回合建议；悬停可查看完整招式循环。
- 事件选择预估已接入：洞府、药圃、灵脉、古龛、残炉、雷池、调息、坊市和修炼选项悬停时会按当前血量、灵石、虚弱、心魔、牌组厚度、试炼签和构筑契合给出收益/风险提示。
- 局外传承已接入：每局结算获得修为，累计修为解锁火弹符、清心诀、灵石催发作为额外起始牌。
- 连胜记录已接入局外存档：通关会提高当前连胜并刷新最高连胜，失败会清空当前连胜；标题页、传承详情、结算页和最近战绩会显示连胜状态。
- 传承解锁路径已扩展：镇魂符、雷火连剑、雷纹护体加入中后期修为解锁，通关后的累计修为仍能持续改变起始牌组。
- 传承 UI 暂用文字呈现；后续可补“外门玉册/传功竹简”背景，用于标题页显示累计修为、通关数和下一解锁。
- 传承页目标摘要已接入：显示下个传承解锁差距、推荐追踪的未完成成就和最近最佳战绩，强化局外长期目标感。
- 标题页已接入传承详情、玩法说明和重置传承入口，方便测试局外存档和向玩家解释核心规则。

- 标题页已接入图鉴入口，可查看卡牌、法宝、流派、劫数、月相、试炼签、悬赏令、本局印记和路线节点；长内容使用滚动面板呈现。

- 成就与挑战详情已增强：未完成条目会显示当前进度或下一步建议，挑战卷轴会提示试炼签缺口、流派通关、黑煞流派、甲等流派和结局卷轴进度。
