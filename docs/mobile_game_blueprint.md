# HeroesLive / 青岚夜行：移动端卡牌 RPG 骨架

## 产品定位

原创国风修仙叙事卡牌 RPG。玩家在五章主线中调查“命册缺页”事件，通过剧情选择、路线探索、职业卡组构筑和跨局成长推进人物关系与世界真相。

参考《月圆之夜》的单手竖屏、选择式叙事和轻量构筑节奏，但角色、世界观、文本、美术与系统命名保持原创。

## 核心循环

山门主城 → 选择章节与职业 → 剧情抉择 → 分支路线 → 战斗/奇遇/坊市/精英 → 首领 → 章节结算 → 解锁新章与永久成长。

每章包含：

- 3—5 个关键剧情场景
- 4 层路线节点
- 普通战、精英战与章节首领
- 至少 2 个影响后续对话或事件的选择
- 卡牌奖励、法宝与成长资源
- 完整的开端、冲突、揭示与章末悬念

## 职业与构筑

| 职业 | 核心资源 | 主要玩法 |
| --- | --- | --- |
| 青竹剑修 | 剑势 | 连击积蓄、终结爆发 |
| 赤篆符师 | 符印 | 延迟附着、集中引爆 |
| 百草丹师 | 药性 | 毒疗转换、寒热调和 |
| 山海御灵 | 灵契 | 灵兽切换、指令协同 |
| 天工偃师 | 机巧 | 机关部署、阵列升级 |
| 幽冥契师 | 魂灯 | 生命交易、弃牌复用 |

当前数据层包含 120 张职业卡牌，每职业 20 张；同时提供 108 套推荐构筑配方。每张牌具备费用、类型、稀有度、关键词、主效果、条件联动和精研方向。战斗使用真实抽牌堆、弃牌堆、消耗区与洗牌循环。

## 五章故事

1. 雨入青岚：外门名册多出一个名字，失踪师姐寄来血书。
2. 灯照玄阴：山道鬼灯写着玩家姓名，旧案开始指向师门。
3. 雷云问心：筑基雷劫被人为改写，主角身世与阵法相连。
4. 黑莲照夜：无灯城居民失去梦境，影子被黑莲教收集。
5. 天门无月：抵达太虚天门，决定修复、焚毁或重写命册。

## 长期成长

- 账号等级与章节解锁
- 悟道树：生命、资源、事件与初始构筑强化
- 职业熟练度：每次选择职业和获得战利牌都会积累，后续用于解锁专属卡牌和开局遗物
- 藏经阁：卡牌发现、升级形态和推荐构筑
- 异闻录：人物关系、线索、敌人和多结局收集
- 每日异闻与挑战种子
