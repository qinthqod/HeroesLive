# HeroesLive / 青岚夜行：移动端卡牌 RPG 骨架

## 产品定位

原创国风修仙叙事卡牌 RPG。玩家在六章主线中调查“命册缺页”与“归墟月潮”事件，通过剧情选择、路线探索、职业卡组构筑和跨局成长推进人物关系与世界真相。

参考《月圆之夜》的单手竖屏、选择式叙事和轻量构筑节奏，但角色、世界观、文本、美术与系统命名保持原创。

## 核心循环

山门主城 → 选择章节与职业 → 剧情抉择 → 分支路线 → 战斗/奇遇/坊市/精英 → 首领 → 章节结算 → 解锁新章与永久成长。

每章包含：

- 3—5 个关键剧情场景
- 4 层路线节点
- 普通战、精英战与章节首领
- 至少 2 个影响后续对话或事件的选择
- 卡牌奖励、法宝与成长资源
- 完整的开端、冲突、揭示与章末悬念

## 职业与构筑

| 职业 | 核心资源 | 主要玩法 |
| --- | --- | --- |
| 青竹剑修 | 剑势 | 连击积蓄、终结爆发 |
| 赤篆符师 | 符印 | 延迟附着、集中引爆 |
| 百草丹师 | 药性 | 毒疗转换、寒热调和 |
| 山海御灵 | 灵契 | 灵兽切换、指令协同 |
| 天工偃师 | 机巧 | 机关部署、阵列升级 |
| 幽冥契师 | 魂灯 | 生命交易、弃牌复用 |

当前数据层包含 120 张职业卡牌，每职业 20 张；同时提供 108 套互不重复的五卡核心构筑。每个职业的 12 张起始牌组由 7 种基础术法按职责组成复数，不再预先塞满全部基础牌或真解牌。藏经阁内可按职业浏览流派图谱，查看入门、进阶、真传梯度、已发现组件、缺失数量、关键词线索与成卷后的出牌思路。战后奖励会识别当前牌组最接近的未完成流派，三选一至少提供一张可推进该图谱的核心牌，并说明契合理由。每张牌具备费用、类型、稀有度、关键词、主效果、条件联动和精研方向。战斗使用真实抽牌堆、弃牌堆、消耗区与洗牌循环。

## 六章故事

1. 雨入青岚：外门名册多出一个名字，失踪师姐寄来血书。
2. 灯照玄阴：山道鬼灯写着玩家姓名，旧案开始指向师门。
3. 雷云问心：筑基雷劫被人为改写，主角身世与阵法相连。
4. 黑莲照夜：无灯城居民失去梦境，影子被黑莲教收集。
5. 天门无月：抵达太虚天门，决定修复、焚毁或重写命册。

## 长期成长

- 账号等级与章节解锁
- 悟道树：生命、资源、事件与初始构筑强化
- 职业熟练度：每次选择职业和获得战利牌都会积累；25/50/75/100 熟练分别解锁额外资粮、核心牌真解、职业资源开局和本命法宝
- 藏经阁：卡牌发现、升级形态和推荐构筑
- 异闻录：人物关系、线索、敌人和多结局收集
- 每日异闻与挑战种子

## 当前纵向切片

已实现主城、先角色后章节的新局流程、六职业选择、每职业 20 张专属技能与 12 张起始牌库、120 张差异化卡牌、108 套构筑配方、每章三段开场剧情、会改变资源或牌组的章节抉择、职业专属事件回应、每章独立的四层路线与敌人曲线、真实抽弃牌循环、抽牌动效、六套职业资源循环、职业熟练度、统一胜利结算、构筑契合推荐、可交互坊市、净心删牌、卡牌精研、调息与修炼节点、保底三选一与跳过奖励、11 件拥有真实被动的法宝、四类战斗消耗品、敌人三式行动循环与下一式预告、章节结算、失败复盘、藏经阁、法宝与命途印记异闻录、会真实影响新局数值的悟道树和本地自动存档。

六职业的运行时状态已接入战斗 UI：剑势、符印、寒热药性、灵兽契约、机巧机关与魂灯均会随出牌变化，并影响伤害、防御、持续效果或回合结算。

职业熟练度已形成四档传承：25 熟练提供额外灵石，50 熟练将职业核心牌替换为真解，75 熟练让每场战斗带着 1 点职业资源开局，100 熟练携带对应本命法宝。职业选择页会同时展示当前已解锁传承和下一目标。

卡牌联动反馈已接入战斗 UI：满足条件的卡牌会以旧金“联”印高亮，单击卡牌立即施放，结算时展开本次触发的伤害、护盾、抽牌与职业资源明细。60 个基础技能模板均具有独立运行时规则或明确的稳定效果，真解卡严格按精研说明改变数值或费用。

本局剧情状态与跨局异闻记录已经分离：章节选择只影响当前云游与当前结局，完成后再进入永久异闻录。路线页持续展示最近的“命途回响”，事件中写明的资源与战斗代价会在后续节点真实兑现并随自动存档恢复。

六章各有两个可收集人物后记，共 12 种。后记回应第二次关键抉择对沈砚秋、陆观、替灯人、无灯城民、守页人或归墟众生造成的长期影响，并写入最近战绩与异闻录；玩家重走另一选择时可补全同章另一篇后记。

首场战斗使用不暂停操作的三步引导：阅读当前与下一招式、单击卡牌立即施放、结束回合。引导一次只聚焦一个动作，可随时跳过，完成后不再重复出现。

六章首领均拥有独立的第一相与半血第二相，每相各含“序—破—急”三式。第七盏灯显露命火灯芯，写名鬼灯释放旧名，雷池守阵者露出阵眼真身，无影城主承受万梦醒转，守门真君显出执笔法相，月蚀司命则会在归墟睁眼并把未行之路写成心魔。当前阶段、当前招式与下一式均提前显示。

每章第二次关键抉择会被对应首领记住，并在半血转相时触发专属台词与战斗后果。守护、偿愿、释放或承担类选择会召回人物关系，为玩家提供护盾；揭露、拒绝、斩断或夺回类选择会破坏首领赖以维持的规则，削减其二阶段护体。六章共形成 12 条可验证的“选择—回应—战术收益”分支。

十场普通与精英战也已从两套公共模板拆分为三十个独立招式。每章普通敌人先教授一种核心压力，精英敌人再把该压力与护体、连击、恢复或牌堆污染组合起来。战场会显示敌人的战斗原型、核心特性和简短反制方向，使章节差异不仅存在于名称和血量中。

失败后会记录当前章节的连续受挫次数，并在下次重启本章时提供至多三级山门扶助：额外石肤符、少量灵石与清神粉。扶助内容在失败页公开，不修改敌人伤害或生命，通关后立即清零。开始新局不再直接增加职业熟练度，避免通过反复开局或故意失败刷永久成长。

山门首页新增挑战卷摘要：从章节通关、百术收录、法宝录、调查宗卷、流派图谱、职业传承和甲等评阶中挑选最接近完成的两项展示。章节通关会写入最近 6 局战绩，记录章节、职业、评阶、分数、牌组规模、法宝和线索，回到山门即可看到本局留下的长期价值。

挑战卷现已形成完整领取闭环：达成目标后优先出现在山门，进入“待领取”状态，单击后一次性结算灵玉、悟道、修为与纪念称号。领取结果写入永久存档锁，重复点击、重新打开异闻录或重启页面都不会再次发放；最新称号会显示在山门顶部，成为长期成长的可见身份反馈。

每日试炼现已成为独立可玩模式：每天按本地日期生成固定职业、章节、异兆和种子；相同种子会复现开局洗牌、重洗、坊市货架、法宝与战利选择。异兆同时提供一项助力与一项真实劫数，例如额外阴雷子对应敌人每段伤害提高。每日首胜一次性奖励灵玉、悟道、修为和“今夜破局”称号，日期锁会阻止重复结算；同日仍可反复挑战以比较构筑与评阶。

挑战复刻将种子扩展为可分享玩法：最近战绩可生成带版本、章节、职业、异兆、种子和校验位的挑战码。导入时先校验内容是否被篡改，再展示完整预览；复刻局不继承原玩家的剧情选择，也不会领取每日首胜。存在未完成云游时禁止覆盖存档，必须先继续或主动放弃。

战斗手感层已覆盖出牌、命中、护体、治疗、抽牌、敌方行动、受击和战利奖励；支持 Web Audio 的设备播放轻量音效，支持振动的移动设备提供触觉反馈。设置中可以分别关闭音效和触觉并调整音量。

整局会记录出牌数、造成伤害、承受伤害、总回合、胜利场次与实际成长收益。章末根据余命、节奏、承伤和完成度动态生成甲上至丙级评阶，并展示真实修为、悟道、灵玉和灵石结余。

藏经阁使用跨局发现进度：各职业起始牌作为初始卷册，后续战利牌、事件牌、坊市牌和真解牌在进入本局牌组时永久收录。未知牌保留稀有度、类型和关键词线索，同时隐藏名称、插图与完整规则。

主城“今夜异闻”会读取真实剧情节点进度。完成第一章三段开场剧情后，一次性获得 8 点悟道并解锁《砚秋手札·雨亭残页》；手札与未完成进度均在异闻录中持续可见。

局内流派目标现已贯穿路线、战斗与牌组卷册：路线节点显示当前构筑方向，战斗 HUD 仅保留名称和五卡进度以控制认知负荷，牌组卷册展示完整打法、下一核心与费用结构。真解牌按同名术法家族计入完成度，精研不会让已形成的流派倒退。

六章均拥有独立调查链：五幕开场剧情给出首条线索与两次抉择，四层路线中的途中剧情、战斗、异闻、精英和坊市分别揭示不同证据，安全的调息或修炼路线则可能放弃证据。调查状态会随本局存档恢复，战斗开场回应最近线索，章末根据证据数量区分“真相已明”或“疑云未尽”，鼓励重返章节探索另一条路线。

调查证据采用“待查证 → 已带回”两阶段结算。选择节点只登记调查目标，战斗获胜、完成异闻或离开坊市后才正式加入线索卷；中途失败不会凭空获得证据，失败页会显示本次未能带回的线索。待查证状态同样进入自动存档，继续战斗后仍可正常完成调查。

异闻录收录六章永久调查宗卷。单局收集 4 条证据即可得到完整章末结论；每章共有 7 条跨分支证据，已带回内容会跨局合并。补齐某章全部证据后，宗卷一次性奖励 12 点悟道与 80 灵玉，重复通关不会重复领取。

六章奇遇与途中剧情均为独立内容：原五章事件之外，归墟新增“月海遗舟”，围绕沈砚秋未曾拥有的人生、沉月龙骨与自由后的遗憾展开。每章奇遇四个选项覆盖构筑、生存、高收益代价和谨慎离开；途中剧情另有两个带真实后果的人物抉择。

六章坊市拥有独立经济结构：在原五处市场之外，“月下浮市”偏向净心、抽牌与终局循环，并允许忘却一张高费牌来偿还未行之路、恢复生命。各市场的购牌、精研、忘却和法宝价格不同，商品同时展示对当前流派的契合理由。

局内新增“试炼札记”：地图、战斗、奇遇与战利页会把本章调查目标、当前路线或敌人、流派进度、长期挑战卷与血线/构筑建议合并展示。札记不使用固定文案，而是从待带回线索、已获证据、敌人反制、当前牌组短板和最近长期目标推导，帮助玩家理解“这一步为什么值得做”。移动端战斗页只显示降噪版札记，避免遮挡手牌、灵气和结束回合。
