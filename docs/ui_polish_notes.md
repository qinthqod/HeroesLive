# UI Polish Notes

## 2026-06-18

- Preserve the ink-navy, jade, parchment, cinnabar and lantern-amber visual language.
- At 720p, prioritize readable hand-card rules text over an oversized battlefield.
- Keep the battlefield, hand and bottom actions visible together without scrolling.
- Use restrained shadows, gold focus rings and subtle scale feedback to clarify interactive surfaces.
- Reserve amber for recommended or primary actions; use jade for navigation and utility.
- Treat the title screen as a cinematic run-setup surface, not a flat directory of every feature.
- Main-page hierarchy is: narrative and primary actions on the left, run configuration in the center, character art on the right, five grouped utility destinations below.
- Secondary destinations belong inside the 图鉴、传承、战绩 category hubs so the first viewport stays calm and game-like.
- Route, event and reward surfaces should always use real raster artwork; missing production art falls back to the closest existing card/enemy asset rather than an empty panel.
- Atmosphere is applied through one shared backdrop shader so future 16:9 production backgrounds inherit the same ink, vignette and rain treatment.
